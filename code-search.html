§141.0302 Zoning Code

A Companion Unit is a detached or attached accessory structure on a residential lot that provides living facilities for one or more persons, independent of the primary dwelling unit, which includes permanent provisions for living, sleeping and cooking. A Junior Liner is an accessory structure that is contained entirely within an existing, habitable single dwelling unit. General Regulations for Companion and Junior Units can be found in SDMC Ch. 14, Article 01, Division 03.
